# Rockbuster-Data-Analysis

## Project Overview
Performed end-to-end data analysis using MySQL's well-known Sakila database.

Distilled actionable business insights for a fictional video rental store using PostgreSQL to query a relational database.

## Key Questions

* Which movies contributed the most/least to revenue gain?
* When do our customers rent movies? Are there identifiable trends in sales figures over time?

## Data and Methodology
The Sakila database is a standard schema that models a DVD rental store. It features tables containing information on films, actors, film-actor relationships, and many other dimension and fact tables relevant to the business, including a central inventory table that connects films, stores, and rentals.

* Wrote comprehensive data dictionary
* Conducted thorough exploratory data analysis
* Wrote complex SQL queries to answer business questions
* Developed compelling business narrative and arrived at clear, actionable insights based on SQL results
* Created high-quality, impactful visualizations and presentation of SQL results
